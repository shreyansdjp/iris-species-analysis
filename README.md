# Iris Species Analysis
## Dataset
The analysis uses the [Iris dataset from Kaggle](https://www.kaggle.com/datasets/uciml/iris).

## Overview
This project analyzes the Iris flower dataset to classify species based on physical measurements.

## Setup

### Prerequisites
- Python 3.8+
- [uv](https://github.com/astral-sh/uv) package manager

### Installation
```bash
uv sync
```

## Usage
```bash
uv run python main.py
```
